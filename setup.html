Timer

A periodic timer that starts a certain point after the start of the game to indicate an action should be taken.

For example, if you wanted a reminder every 10 minutes after 1h30m, use 90 for Start time, and 10 for Interval.

A start time of 0 enables the timer immediately. An interval of 0 disables the timer.

Board Layout

The Pokemon set field is fully customizable with Pokemon numbers. Use {gen#} to include all the Pokemon from a certain gen. For example, use {gen3},{gen4} for a board containing only generation 3 and 4 Pokemon.

Multiple numbers can be specified using commas. Ranges can be specified using a hyphen. For example, use 1,4,7,147-151 for a board containing only the generation 1 starters and legendaries.

{gen#} macros and numbers can be combined, but must be separated by commas: 1,4,7,147-151,{gen3}.
